# Financial Kill Line (Execution Line) Calculator
A simple web tool to assess personal financial "kill line" (execution line) risk, tailored for users in **Hong Kong** and the **United States**.

## 📌 Project Inspiration
This project was inspired by the trending online discussion around the "financial kill line" (a colloquial term referring to the critical financial threshold where individuals face irreversible financial distress). The tool focuses on region-specific financial metrics for Hong Kong and the US, reflecting the unique economic contexts of both regions.

## 🌍 Target Users
- Hong Kong residents (with metrics like negative equity, domestic helper expenses, private school costs)
- US residents (with metrics like 400 USD emergency fund threshold, medical/unemployment insurance, debt-to-income ratio)

## ✨ Core Features
- Region-specific risk assessment (HK/US)
- Dynamic form fields based on selected region
- Instant risk result visualization (high/medium/low risk)
- "Zoey" easter egg: Special pass for users named Zoey (100% healthy financial status)
- Responsive UI (works on mobile/desktop)

## 🛠️ Tech Stack
- HTML + Tailwind CSS (UI/Layout)
- Vanilla JavaScript (Core calculation logic)
- No backend required (fully client-side)

### Option 1: Local Use
1. Clone this repo:
   ```bash

   ```
2. Open `execution_line.html` (or `index.html` if renamed) in any modern browser
